Como interpretar estes dados?

Média de Vibração: Representa a suavização do ruído geral da máquina. O aumento constante desta linha indica desgaste progressivo de componentes.

Impactos Máximos (Picos): Representam os "solavancos" mais fortes que a máquina sofreu.

Como este relatório é gerado?

Fonte de Dados: O sistema utiliza o banco de dados principal dados_sensor.db, que contém todo o histórico coletado em tempo real pelo Arduino.

Algoritmo de IA: Utilizamos o Isolation Forest (Floresta de Isolamento). É um algoritmo de Machine Learning especializado em detectar anomalias sem precisar de exemplos prévios de falhas.

O que ele busca? Ele analisa simultaneamente a média móvel, o desvio padrão e os picos de vibração. Pontos que se afastam drasticamente do padrão normal da máquina são "isolados" e marcados como anomalias (os 'X' vermelhos no gráfico).
